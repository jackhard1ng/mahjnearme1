# Directory submission checklist

Each of these is a free backlink plus a separate channel for traffic. Prioritize
the top section — those move the needle. The lower sections are nice-to-haves
once the top is done.

Mark completion as you go: `[x]`

---

## Tier 1 — Do first (high-value backlinks + direct traffic)

- [ ] **Google Business Profile** — register MahjNearMe as a business. Local
      SEO multiplier.
- [ ] **Bing Places** — same. Takes 5 minutes.
- [ ] **Apple Business Connect** — Apple Maps searches go through this.
- [ ] **Google Search Console** — submit sitemap, verify ownership.
- [ ] **Bing Webmaster Tools** — submit sitemap.
- [ ] **Yandex Webmaster** — small but free. Two minutes.
- [ ] **DuckDuckGo** — auto-pulled from Bing, so no action needed beyond Bing.

## Tier 2 — Board game / hobby directories

- [ ] **BoardGameGeek** — MahjNearMe as a resource in the Mahjong entry
      (community-edited). Reach out in the Mahjong forum first.
- [ ] **Meeple Like Us** — accepts resource submissions.
- [ ] **Hobby Search Engine** (hobbysearch.net) — paid but inexpensive listing.
- [ ] **r/Mahjong wiki** — message mods to add MahjNearMe to the sidebar.
- [ ] **Riichi Reddit wiki** — same.
- [ ] **American Mah Jongg Association** — if they maintain a directory,
      request inclusion.

## Tier 3 — Local / "Things to Do" directories

For each major metro (NYC, LA, Chicago, Miami, DC, Boston, SF, Seattle, Dallas,
Houston, Atlanta, Philadelphia):

- [ ] **City's official "Things to Do" site** (e.g., nycgo.com, visitdallas.com)
- [ ] **Timeout [City]** — reach out to the events editor with a pitch
- [ ] **Local subreddit weekly thread** — `r/AskNYC`, `r/LosAngeles`, etc.
      (follow self-promo rules — contribute for weeks before posting a link)
- [ ] **Local Patch site** — they accept press releases for free
- [ ] **Local senior-community newsletters** — email the activities coordinator

## Tier 4 — Event aggregators

- [ ] **Meetup** — create a free MahjNearMe group page that links back to the
      site. Don't mirror listings, just use it as a distribution channel.
- [ ] **Eventbrite** — if you're ever running an event, list it here with
      MahjNearMe as the organizer.
- [ ] **Facebook Events** — same, for any in-person party or launch.
- [ ] **AllEvents.in** — free event listing site with decent SEO.

## Tier 5 — Senior-adjacent directories

Mahjong over-indexes with 55+ women. These have low traffic but high conversion.

- [ ] **AARP community resources** — submit via their local-events form.
- [ ] **Senior Advisor / A Place for Mom** — activities directories for
      independent-living communities.
- [ ] **Road Scholar** (formerly Elderhostel) — mahjong-related programs
      sometimes list partner resources.
- [ ] **NextDoor** — post in each of your personal neighborhood feeds. Do NOT
      spam multiple cities from one account; they'll shadowban.

## Tier 6 — Jewish community directories

Mahjong has deep, active roots in Jewish-American communities.

- [ ] **JCC Association of North America** — request inclusion in their
      resource directory.
- [ ] **Jewish Federations of North America** — local-chapter resource lists.
- [ ] **Chabad.org** event directories by city.
- [ ] **My Jewish Learning** — occasionally links to community resources.

## Tier 7 — General web directories (low priority, only if you have time)

- [ ] **DMOZ descendants** — Curlie.org, Jasmine Directory
- [ ] **Best of the Web** (paid)
- [ ] **Business.com** (paid)

Modern SEO barely weights these — do them last or skip.

---

## Tracking

Keep a spreadsheet with: directory name, submission date, status, follow-up
date, URL of listing (once approved). Use this as raw material for future
outreach.

## Cadence
- Tier 1: one day, do all at once.
- Tier 2–3: one per day for two weeks. Don't batch — spacing them out makes
  them look more organic to Google.
- Tier 4–5: as needed.
- Tier 6–7: optional.
